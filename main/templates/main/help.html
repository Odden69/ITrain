This is how you use ITrain

To use ITrain efficiently you need to build your own stock of Exercises and Workouts.
Start by adding the Exercises you like best, like lunges and squats for instance, and put them together in Workouts. Then you'll soon figure out how ITrain works.

Check out the instructions below for more detailed information

Exercises

  

The Exercise list:
  •  :  Click to see more information about the exercise.
  •  :  Click to edit the exercise.
  •  :  Click to delete the exercise
Fill out the Exercise form like this:
  • Name: Give the Exercise a clear name, to separate similar exercises.
  • Unit: If the exercise involves weights, select kg. If not, select -.
  • Muscle Group: Select all the muscle groups the exercise uses.
    If you can't find the muscle group you need, add one under the Muscle Groups menu.
  • Description: This is your app, write something you might need in the gym when doing this exercise

Workouts

  

The Workout list:
  •  :  Click to see more information about the workout.
  •  :  Click to edit the workout.
  •  :  Click to delete the workout
Fill out the Workout form like this:
  • Name: Give the Workout a clear name, to separate similar workouts.
  • Description: This is your app, write something you find useful.
  • Add Exercise: Push the button to add another exercise to the workout.
  • Exercises: Select one exercise from the list.
  • Value: Use this field for the weight if your exercise has kg as a unit.
  • Reps unit: If you count your exercises, select reps. If you do them for, say 1 min, select min.
  • Reps: Enter how many reps you will do (or minutes, if you chose that option)
  • Sets: Enter how many sets you will do
  • Delete: Check this box if you want to delete the exercise from the workout. When you click Save, the exercise will be removed.
    If the exercise is left empty you don't need to check the box, it will be deleted anyway.

Sessions

  

The Calendar:
  • Create New Session: Click to create a new session.
  • Previous: Click to go to the previous month.
  • Next: Click to go to the next month.
  • Click on a session to view it.
The Session:
  •  :  Click to edit the session.
  •  :  Click to delete the session
  • This is the page you use at the gym!
Fill out the Session form like this:
  • Name: Give the Session a good name.
  • Date: Pick the date for the session.
  • Workout: Select as many workouts as you like.
  • Exercises: Select one exercise from the list.
  • Comment: Make a comment on the workout.

Muscle Groups

  

The Muscle Group list:
  • Create New Muscle Group: There are lots of provided muscle groups, but you can add your own here
  •  :  Click to edit the muscle group.
  •  :  Click to delete the muscle group.
